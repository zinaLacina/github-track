Introduction

github-track is a Python library to use the Github API v3. With it, you can pull any public repositories pull requests from Python scripts.

Short tutorial

First create a GhTrack instance:

from ghtrack import GhTrack

# using an access token
g = GhTrack("access_token")

# Using default values
g = GhTrack()

Then play with your Github objects:

for pull in g.getPulls():
    print(pull["title"])

Download and install

This package is in the Python Package Index, so pip install ghtrack should be enough. You can also clone it on Github.
